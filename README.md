# Rice Renewable Energy Fund — Analytics Project

This project is being built for the **Rice Renewable Energy Fund (RNEF)**, a student-run investment fund at Rice University that focuses on renewable energy companies.

The goal is to give the fund team a set of tools to **track their portfolio** and **test investing strategies** — so they can make smarter decisions about what to buy and sell.

---

## What's in this folder

### PDF Files

- **RNEF_Fund_Ideas.pdf** — A document with early ideas and brainstorming about what tools and features to build for the fund.
- **RNEF_Dev_Timeline.pdf** — A rough plan and timeline for how the project will be developed over time.

These are great starting points if you want to understand the big picture before looking at any code.

---

## Folders

### `rnef-backtest/`

This is where the main web application lives.

In simple terms: it's a **website** that lets the fund team:

1. See their current stock holdings in a clean dashboard
2. Set up a "backtest" — which means testing a strategy on *past* data to see how it would have done
3. View charts and stats from those tests

There is a separate README inside that folder that explains how to run it and how everything is organized. Start there if you want to work on the code.
